# nsscreencast-fetcher

[A Clojure example that downloads screencasts.] (http://myotherpants.com/2014/01/are-you-doing-anything-real-with-it/)

## Usage

Add your secret key to NSScreencast's private rss feed then...

    lein deps
    lein run
